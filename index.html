Fitur Utama
Platform lengkap untuk
setiap tahap analisis

Dari upload data hingga interseksi multi-layer dan kolaborasi tim — semua tersedia dalam satu alat.

+
Tambah Layer Kustom

Definisikan layer baru dengan tipe geometri (polygon, titik, garis, raster, mesh 3D), atribut, dan aturan validasi sendiri. Layer langsung muncul sebagai overlay di peta Cesium.

Baru
⇳
Upload Multi-Format

Upload data spasial dalam format GeoJSON, Shapefile, GeoTIFF, CSV (titik), KML, glTF (mesh 3D), hingga LiDAR LAZ. Validasi atribut otomatis sesuai rule yang Anda definisikan.

5 Format
△
Intersection Type Explorer

Lihat tipe geometri hasil interseksi dua layer: polygon ∩ titik = titik, garis ∩ garis = titik, polygon ∩ garis = garis. Matriks referensi tersedia di panel Interseksi.

Geometri
📊
Skor CIAS Multi-Layer

Coconut Industrial Attractiveness Score kini merangkum layer sistem dan layer kustom Anda. Setiap grup layer (misalnya "Lingkungan") mendapat skor CIAS tersendiri.

CIAS
⨁
Layer Groups

Kelompokkan layer-layer terpilih menjadi satu grup tematik (cth: Lingkungan, Sosial Ekonomi, Infrastruktur). Multi-select di sidebar, lalu satu klik buat grup.

Organisasi
👤
Autentikasi Institusi

Daftar dengan institusi/grup Anda (Kementan, IPB, swasta, dsb.). Layer yang dibagikan (shared) hanya dapat diakses oleh anggota institusi yang sama — privasi data terjaga.

Kolaborasi
🗺
Peta Cesium 3D Interaktif

Globe 3D berbasis Cesium.js dengan overlay layer vector dan raster, hover tooltip, mode bar 3D, dan kontrol zoom/tilt. Fallback SVG 2D jika token tidak tersedia.

Cesium
🔬

Bangun query "WHERE pH > 5.5 AND curah hujan > 1500mm AND elevasi < 600m" — hasil divisualisasikan di peta dan dirangkum dalam skor CIAS gabungan.

Intersection Type Explorer

Tahu persis tipe geometri
hasil interseksi Anda

Saat Anda menginterseksikan dua layer dengan tipe geometri berbeda, platform otomatis menghitung dan menampilkan tipe data hasil — polygon, garis, atau titik.

Prinsip dimensi: data dengan dimensi lebih tinggi yang diinterseksikan dengan dimensi lebih rendah menghasilkan data dimensi lebih rendah.

⬡ Polygon ∩ ⬤ Titik = ⬤ Titik 〰 Garis ∩ 〰 Garis = ⬤ Titik ⬡ Polygon ∩ 〰 Garis = 〰 Garis ▦ Raster ∩ ⬤ Titik = ⬤ Sample
Matriks Tipe Hasil Interseksi
∩ ▦ Raster ⬡ Polygon 〰 Garis ⬤ Titik
▦ Raster ▦ Raster ▦ Clip 〰 Profil ⬤ Sample
⬡ Polygon ▦ Clip ⬡ Polygon 〰 Garis ⬤ Titik
〰 Garis 〰 Profil 〰 Garis ⬤ Titik ⬤ Titik
⬤ Titik ⬤ Sample ⬤ Titik ⬤ Titik ⬤ Titik

Matriks ini tersedia langsung di panel Interseksi → tab "Tipe Geometri Hasil"

Pengguna
Satu platform, empat persona

Fitur dan akses data disesuaikan dengan profil institusi yang terdaftar.

🏧 Pemerintah
Kementan / Pemda / BRIN

Upload dan kelola dataset resmi. Tetapkan threshold kesesuaian nasional. Bagikan ke institusi pemerintah terkait.

  • Upload dataset resmi (GeoTIFF, SHP)
  • Kelola threshold kesesuaian wilayah
  • Ekspor laporan kebijakan
  • Akses layer institusi pemerintah lain
  • Buat layer group tematik nasional
💰 Investor & Swasta
Investor / Perusahaan

Evaluasi lokasi investasi, simulasi ROI, dan identifikasi kabupaten dengan CIAS tertinggi untuk derivatif target.

  • Ranking CIAS per kabupaten/derivatif
  • Simulasi skenario kebijakan
  • Komparasi hingga 4 lokasi
  • Upload data internal (private layer)
  • Due diligence dengan data resmi
🔬 Peneliti
Peneliti / Akademisi

Upload dataset riset, definisikan atribut validasi kustom, jalankan analisis interseksi multi-kriteria, dan ekspor data mentah.

  • Tambah layer kustom + validasi
  • Upload GeoTIFF, GeoJSON, CSV, mesh
  • Interseksi multi-layer dengan CIAS
  • Ekspor GeoJSON / CSV hasil analisis
  • Bagikan layer ke tim riset institusi
🏠 Tim / Institusi
Tim Antar-Institusi

Anggota institusi yang sama dapat mengakses layer shared, berkolaborasi dalam satu layer group, dan melihat kontribusi data rekan.

  • Akses layer shared institusi
  • Kolaborasi dalam layer group
  • Lihat siapa yang upload tiap layer
  • Toggle on/off layer tim dengan mudah
  • CIAS gabungan layer tim & sistem
